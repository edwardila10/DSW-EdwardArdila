Etiquetas Básicas de HTML, atributos y valores

Propósito del sitio

Queremos con esta página divulgar información acerca de las características básicas de HTML, con sus respectivos atributos y valores, mostrando la forma de como implementarlas en documentos HTML.La tecnología HTML es que los caracteres de control (etiquetas), se insertan en un documento de texto sin formato y como resultado obtenemos una página web. Al cargar una página web, un navegador (por ejemplo, Internet Explorer) la presenta en la pantalla en la forma especificada por las etiquetas.

En resumen, el HTML es un lenguaje de etiquetas en donde se marcan los espacios en los que va a ir el contenido. Para ello, se describen las estructuras básicas de una determinada página y se pueden incluir enlaces a otros sitios. En otras palabras, podríamos decir que sirve para crear webs y darle una determinada estructura y contenido. El HTML es fundamental para el normal funcionamiento de un sitio web, ya que un error en el enlace puede hacer que nuestros usuarios tengan una mala experiencia navegando en la tienda. Aprender sobre HTML, es el primer gran paso para todos aquellos que deseen convertirse en desarrolladores web porque estamos hablando de, ni más ni menos, el principal lenguaje de marcado de Internet. Es importante entender que cada página HTML, contiene una serie de elementos que crean la estructura del contenido de un sitio o aplicación. También es clave tener siempre presente que se utiliza, principalmente, para crear web estáticas. Es por eso que es fundamental relacionar el HTML con la CSS y JavaScript para lograr una página dinámica para que sea más atractiva para los usuarios.
